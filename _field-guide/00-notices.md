**What to expect from the pilot:** Testing phase will a stable clustered `CryoSPARC` and `Relion`

For support inquiries email [sbf+discworld@unsw.edu.au](mailto:support.discworld@sbf.unsw.edu.au), please **read the guide first**.
{: .notice--info}

We can trial these cluster features because we have a new system, and a small number of users entirely focused on CryoEM *processing* for long-term engagements. It does not operate in a way that allows more general purpose use. General purpose cluster serve a different purpose, so have different functions and problems to solve. **Note:** if using our user-friendly helper commands, direct `command` or `script` portability will <u>NOT</u> transfer to other computer systems. 
{: .notice--success}
